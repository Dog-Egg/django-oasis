Auth¶

class django_oasis.auth.BaseAuth¶

认证基类。如需自定义认证类，需继承该类，并实现其抽象方法。

abstract check_auth(request)¶

需自行实现该方法，用于判断请求认证是否成功。如果认证失败，需要抛出异常来停止请求的继续处理。

参数:

request – Django HttpRequest 对象。

declare_responses: Dict[int, dict]¶

描述认证时会产生的响应。键为 HTTP 状态码，值为 Response Object。

declare_security: dict¶

定义认证使用的安全方案。declare_security 的值为 Security Scheme Object。

备注

以下认证类是建立在使用 django.contrib.auth 包作为项目的用户验证系统。如果你的项目使用了自定义的用户验证，请查看 自定义认证。

class django_oasis.schema.Model(required_fields=None, unknown_fields='exclude', **kwargs)¶
参数:

  • required_fields (Iterable[str] | str | None) –

    覆盖原有的必需字段的配置。设为空列表则所有字段为非必需，也可设为 "__all__" 指定所有字段为必需。

    >>> class User(Model):
...     username = String()
...     address = String(required=False)
    设为 [] 指定所有字段为非必需¶
    >>> User(required_fields=[]).deserialize({})
{}
    设为 “__all__” 指定所有字段为必需¶
    >>> import pprint

>>> try:
...     User(required_fields='__all__').deserialize({})
... except ValidationError as exc:
...     pprint.pprint(exc.format_errors())
[{'loc': ['username'], 'msgs': ['This field is required.']},
 {'loc': ['address'], 'msgs': ['This field is required.']}]
    指定部分字段为必需¶
    >>> try:
...     User(required_fields=['address']).deserialize({})
... except ValidationError as exc:
...     pprint.pprint(exc.format_errors())
[{'loc': ['address'], 'msgs': ['This field is required.']}]

  • unknown_fields (str) –

    该参数决定了在反序列化时如何处理未知字段。

    >>> class User(Model):
...     name = String()
...     age = Integer()

>>> data = {'name': '张三', 'age': 24, 'address': '北京'}
    exclude (默认)¶
    >>> User().deserialize(data)
{'name': '张三', 'age': 24}
    include¶
    >>> User(unknown_fields='include').deserialize(data)
{'name': '张三', 'age': 24, 'address': '北京'}
    error¶
    >>> User(unknown_fields='error').deserialize(data)
Traceback (most recent call last):
    ...
django_oasis_schema.exceptions.ValidationError: [{'msgs': ['Unknown field.'], 'loc': ['address']}]

class django_oasis.schema.Schema(*, required=None, default=undefined, nullable=False, validators=None, choices=None, description='', clear_value=<function default_clear_value>, error_messages=None, after_deserialization=None, **kwargs)¶
参数:

  • required (bool | None) –

    *仅作为字段时有效* 在反序列化时，判断字段是否必需提供。默认必需，可设为 False 或设置 default 参数使其变为非必需。

    >>> class User(Model):
...     username = String()
...     address = String(required=False)

>>> User().deserialize({})
Traceback (most recent call last):
    ...
django_oasis_schema.exceptions.ValidationError: [{'msgs': ['This field is required.'], 'loc': ['username']}]

  • default (Any | Callable[[], Any]) –

    *仅作为字段时有效* 如果字段非必需，且设置了默认值，那么反序列化时如未提供该字段数据，则使用该默认值填充。该参数可设为一个无参函数，默认值将使用该函数的结果。

    >>> from datetime import date

>>> class User(Model):
...     username = String(default='未命名用户')
...     address = String(default=lambda: '未提供')  # 使用函数设置默认值

>>> User().deserialize({})
{'username': '未命名用户', 'address': '未提供'}

  • nullable (bool) –

    执行验证时判断数据是否为 None。默认 False，表示不可以为 None。

    >>> String(nullable=True).deserialize(None) is None
True

    >>> String().deserialize(None)

    ...
django_oasis_schema.exceptions.ValidationError: [{'msgs': ['The value cannot be null.']}]

django_oasis.schema.as_getter(field)¶

自定义 Model 字段序列化时的取值函数。默认是对 Mapping 取索引，对其它对象取属性名。

参数:

field (Schema | str) – 字段或字段名，将取值函数应用于该字段。

>>> from django_oasis import schema

>>> class Person(schema.Model):
...     fullname = schema.String()
...
...     @schema.as_getter(fullname)
...     def get_fullname(self, data):
...         return data['firstname'] + data['lastname']

>>> Person().serialize({'firstname': '张', 'lastname': '三'})
{'fullname': '张三'}
