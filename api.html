class django_oasis.OpenAPI(*, name: str | None = None)#
参数:

name – 如果需要对外分享 OAS 数据，建议设置该名称，它将作为 OAS 数据地址的一部分，而不是使用计算出的名称。

class django_oasis.Operation(*, tags=None, summary: str | None = None, description: str | None = None, response_schema: None | Type[Model] | Dict[str, Schema] | Schema = None, deprecated: bool = False, include_in_spec: bool = True, auth: BaseAuth | Type[BaseAuth] | None = None, status_code: int = 200, view_decorators: list | None = None)#
参数:

  • include_in_spec – 是否将当前操作解析到 OpenAPI Specification 中，默认为 True。

  • auth – 设置操作请求认证。

class django_oasis.Resource(path: str, *, param_schemas: Dict[str, Schema] | None = None, param_styles=None, tags=None, view_decorators: list | None = None, include_in_spec: bool = True, default_auth: BaseAuth | Type[BaseAuth] | None = None)#

  • path – 资源 URL，必须以 “/” 开头。

  • include_in_spec – 是否将当前资源解析到 OpenAPI Specification 中，默认为 True。

  • default_auth – 为所属的 Operation 提供默认的 auth。

class django_oasis.schema.Model(required_fields: Iterable[str] | str | None = None, unknown_fields: str = 'exclude', **kwargs)#
参数:

  • required_fields –

    覆盖原有的必需字段的配置。设为空列表则所有字段为非必需，也可设为 "__all__" 指定所有字段为必需。

    >>> class User(Model):
...     username = String()
...     address = String(required=False)
    设为 [] 指定所有字段为非必需#
    >>> User(required_fields=[]).deserialize({})
{}
    设为 “__all__” 指定所有字段为必需#
    >>> import pprint

>>> try:
...     User(required_fields='__all__').deserialize({})
... except ValidationError as exc:
...     pprint.pprint(exc.format_errors())
[{'loc': ['username'], 'msgs': ['This field is required.']},
 {'loc': ['address'], 'msgs': ['This field is required.']}]
    指定部分字段为必需#
    >>> try:
...     User(required_fields=['address']).deserialize({})
... except ValidationError as exc:
...     pprint.pprint(exc.format_errors())
[{'loc': ['address'], 'msgs': ['This field is required.']}]

  • unknown_fields –

    该参数决定了在反序列化时如何处理未知字段。

    >>> class User(Model):
...     name = String()
...     age = Integer()

>>> data = {'name': '张三', 'age': 24, 'address': '北京'}
    exclude (默认)#
    >>> User().deserialize(data)
{'name': '张三', 'age': 24}
    include#
    >>> User(unknown_fields='include').deserialize(data)
{'name': '张三', 'age': 24, 'address': '北京'}
    error#
    >>> User(unknown_fields='error').deserialize(data)
Traceback (most recent call last):
    ...
django_oasis_schema.exceptions.ValidationError: [{'msgs': ['Unknown field.'], 'loc': ['address']}]

class django_oasis.schema.Schema(*, required: bool | None = None, default: ~typing.Any | ~typing.Callable[[], ~typing.Any] = EMPTY, nullable: bool = False, validators: ~typing.List[~typing.Callable[[~typing.Any], ~typing.Any]] | None = None, choices: ~typing.Iterable | None = None, description: str = '', clear_value: ~typing.Callable[[~typing.Any], bool] | None = <function default_clear_value>, error_messages: dict | None = None, after_deserialization: ~typing.Callable | None = None, **kwargs)#
参数:

  • required –

    *仅作为字段时有效* 在反序列化时，判断字段是否必需提供。默认必需，可设为 False 或设置 default 参数使其变为非必需。

    >>> class User(Model):
...     username = String()
...     address = String(required=False)

>>> User().deserialize({})
Traceback (most recent call last):
    ...
django_oasis_schema.exceptions.ValidationError: [{'msgs': ['This field is required.'], 'loc': ['username']}]

  • default –

    *仅作为字段时有效* 如果字段非必需，且设置了默认值，那么反序列化时如未提供该字段数据，则使用该默认值填充。该参数可设为一个无参函数，默认值将使用该函数的结果。

    >>> from datetime import date

>>> class User(Model):
...     username = String(default='未命名用户')
...     address = String(default=lambda: '未提供')  # 使用函数设置默认值

>>> User().deserialize({})
{'username': '未命名用户', 'address': '未提供'}

  • nullable –

    执行验证时判断数据是否为 None。默认 False，表示不可以为 None。

    >>> String(nullable=True).deserialize(None) is None
True

    >>> String().deserialize(None)
Traceback (most recent call last):
    ...
django_oasis_schema.exceptions.ValidationError: [{'msgs': ['The value cannot be null.']}]

  • description – 在 OpenAPI Specification 中提供描述内容。

  • validators –

    用于设置反序列化验证函数。

    >>> def validate(value):
...     if value <= 0:
...         raise ValidationError('不是一个正整数')

>>> Integer(validators=[validate]).deserialize(-1)
Traceback (most recent call last):
    ...
django_oasis_schema.exceptions.ValidationError: [{'msgs': ['不是一个正整数']}]

django_oasis.schema.serialization_fget(field: Schema | str)#

自定义 Model 序列化时字段取值。默认是对 Mapping 取索引，对其它对象取属性名。

参数:

field – 字段或字段名，将取值函数应用于该字段。

>>> from django_oasis import schema

>>> class Person(schema.Model):
...     fullname = schema.String()
...
...     @schema.serialization_fget(fullname)
...     def get_fullname(self, data):
...         return data['firstname'] + data['lastname']

>>> Person().serialize({'firstname': '张', 'lastname': '三'})
{'fullname': '张三'}

如下是在 Model 中使用 validator 验证日期的正确性。

>>> class MySchema(schema.Model):
...     start_date = schema.Date()
...     end_date = schema.Date()
...
...     @schema.validator()
...     def validate_date(self, data):
...         if data['start_date'] > data['end_date']:
...             raise schema.ValidationError('开始日期不能大于结束日期')

>>> MySchema().deserialize({'start_date': '2000-01-01', 'end_date': '1999-01-01'})

    ...
django_oasis_schema.exceptions.ValidationError: [{'msgs': ['开始日期不能大于结束日期']}]

field –
