请求体#

JSON 格式#

class django_oasis.parameter.JsonData(schema)#

声明请求体数据，要求以 JSON 格式提交。

参数:

schema (Schema | Type[Schema] | Dict[str, Schema]) –

代码示例：

from django_oasis import Resource, schema
from django_oasis.parameter import JsonData


@Resource("/to/path")
class API:
    def post(
        self,
        data=JsonData(
            {
                "a": schema.Integer(),
                "b": schema.String(),
            }
        ),
    ):
        ...

表单格式#

class django_oasis.parameter.FormData(schema, /)#

声明请求体数据，要求以表单格式提交。

参数:

schema (Model | Type[Model] | Dict[str, Schema]) –

代码示例：

from django_oasis import Resource, schema
from django_oasis.parameter import FormData


@Resource("/to/path")
class API:
    def post(
        self,
        form=FormData(
            {
                "a": schema.String(),
                "b": schema.Integer(),
            }
        ),
    ):
        ...

当 POST 表单为 a=1&b=1 时，form 值等于 {'a': '1', 'b': 1}。

class django_oasis.parameter.FormItem(schema)#

这是 FormData 的变体，用于声明表单中的一个键，而不是整个表单。

参数:

schema (Schema | Type[Schema]) –

代码示例：

from django_oasis import Resource, schema
from django_oasis.parameter import FormItem


@Resource("/to/path")
class API:
    def post(
        self,
        a=FormItem(schema.String()),
        b=FormItem(schema.Integer()),
    ):
        ...

当 POST 表单为 a=1&b=1 时，a 值等于 "1"，b 值等于 1。

Body#

class django_oasis.parameter.Body(schema, *, content_type='application/json', description='')#

声明 HTTP 请求体数据，默认数据格式类型为 JSON。

参数:

  • content_type (str | List[str]) – 请求体内容类型，默认是: application/json。也可以设置为列表，以支持多种请求体类型。

  • description (str) –

代码示例：

from django_oasis import Resource, schema
from django_oasis.parameter import Body


class BookSchema(schema.Model):
    title = schema.String(description="书名")
    created_at = schema.Datetime(description="创建时间")


@Resource("/to/path")
class API:
    def post(self, body=Body(BookSchema)):
        ...
